## Query Guide

### Options for Integer and Floating Point Queries

You can prepend integers and floating points with certain strings for better search functionality.

* (equal)
* ! (not equal)
* < (less than)
* <= (less than or equal to)
* \> (greater than) 
* \>= (greater than or equal to)
